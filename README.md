# Rust-Raid-Calculator
 Быстрый и удобный калькулятор рейдов для Rust. Расчет стоимости взрыва стен, дверей и других объектов..

💥 Rust Raid Calculator 
Инструмент:
для быстрого расчета стоимости рейдов в игре Rust. Помогает 
спланировать атаку, рассчитать необходимое количество взрывчатки и 
выбрать наиболее выгодный путь для уничтожения базы.

✨ Особенности:
Полная база объектов: поддержка всех видов стен, дверей, турелей и шкафов.
Выбор взрывчатки: расчет в С4, ракетах, бобовых гранатах, сачелях или разрывных патронах.
Оптимизация ресурсов: вывод точного количества серы, пороха и угля для крафта.
Быстрый расчет: мгновенный результат при вводе количества целей.

🛠 Стек технологий:
Rust (Python 3.12)

🚀 Как запустить:
Прописать в терминал: python main.py

> [!WARNING]
> Проект активно развивается и дорабатывается. 
> Возможны небольшие неточности или ошибки в расчетах.
