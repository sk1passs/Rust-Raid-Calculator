# RUST HELPER

Быстрый и удобный калькулятор рейдов для Rust. Расчет стоимости взрыва стен, дверей и других объектов, а также встроенная база кодов для камер видеонаблюдения.

---

## 💥 Rust Raid Calculator Инструмент
Для быстрого расчета стоимости рейдов в игре Rust. Помогает спланировать атаку, рассчитать необходимое количество взрывчатки и выбрать наиболее выгодный путь для уничтожения базы.

### ✨ Особенности:
* **Полная база объектов**: поддержка всех видов стен, дверей, турелей и шкафов.
* **Выбор взрывчатки**: расчет в C4, ракетах, бобовых гранатах, сачелях или разрывных патронах.
* **Оптимизация ресурсов**: вывод точного количества серы, пороха и угля для крафта.
* **Быстрый расчет**: мгновенный результат при вводе количества целей.
* **База CCTV кодов**: встроенные справочные таблицы с кодами стационарных камер для всех ключевых РТ игры, полностью переведенные на русский язык.

---

## 📋 Добавленная база кодов камер (CCTV)

В утилиту добавлены структурированные списки камер для следующих локаций:
* **Аэродром** — вертолетная площадка.
* **Лагерь бандитов** — казино и магазин оружия.
* **Сфера** — купол снизу и самый верхний ярус.
* **Большая нефтяная вышка** — пошаговый список всех 13 камер по этажам от причала до комнаты связи.
* **Мирный город** — улицы, нефтеперегонка, зоны отдыха и музыкальные сцены.
* **Малая нефтяная вышка** — поэтажный список от лодочного дока до верхней палубы.
* **Подводная лаборатория** — все 9 отсеков (столовая, каюты, коридоры охраны) с сохранением серверных масок.
